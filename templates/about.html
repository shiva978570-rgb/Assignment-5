Project Highlights

What makes this different?

Beyond basic bag-of-words — this analyzer uses a full multi-step NLP pipeline.

🔄

Negation Handling

"not good" → "not_good". Negation tokens are fused with the next word so the model understands reversed sentiment correctly.

😍

Emoji Recognition

Emojis mapped to words before cleaning (😍→love, 😡→angry), preserving emotional signals that text-only models miss.

📝

Sentence-Level Voting

Each sentence is predicted independently. Final result = majority vote using Counter(). Perfect for mixed reviews.

🌱

Lemmatization

WordNet lemmatizer reduces words to base forms (running→run) for better vocabulary generalization.

📊

Confidence Score

Every prediction shows a confidence percentage from predict_proba() so you know how sure the model is.

🌐

Full Web App

Login, dashboard, review history with filters and delete, about page — complete production-ready Flask app.

Development Journey

How It Was Built

📁
Phase 1

Dataset Exploration

Loaded 309,296 Flipkart reviews (CSV, latin1 encoding). Checked nulls, duplicates, rating distribution.

🧹
Phase 2

Advanced Preprocessing

Built full NLP pipeline: emoji replacement → clean_text() → handle_negation() → lemmatize_text().

🏷️
Phase 3

Sentiment Labelling

Rate ≥ 4 → Positive, Rate = 3 → Neutral, Rate ≤ 2 → Negative. Filtered out 'unknown' labels.

🔢
Phase 4

TF-IDF + Model Training

TF-IDF with 5000 max features and bigrams. Trained Naive Bayes and Logistic Regression. Evaluated with classification report.

🗳️
Phase 5

Sentence-Level Voting

predict_review() tokenizes into sentences, predicts each separately, uses Counter().most_common(1) for final vote.

🌐
Phase 6

Flask Web Application

Complete 4-page app: login, dashboard (with sentence breakdown), My Reviews (with filters), About page.
